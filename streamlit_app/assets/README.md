# Application Asset Directory

This directory hosts graphical assets and screenshot previews used in the systems documentation (`README.md`) and procurement manuals.

## Guidelines for Screenshots
When capturing previews for client demonstrations:
1. Capture screenshots using the standard browser zoom (100%).
2. Save screenshots in standard PNG formats using the exact naming convention:
   - `dashboard_preview.png` (Command Center)
   - `historical_preview.png` (Historical Analytics Tab)
   - `inventory_preview.png` (Reorder Planning Tab)
   - `forecast_preview.png` (Next Month Projections Tab)
3. Ensure no private client credentials or vendor details are exposed in the visuals.
